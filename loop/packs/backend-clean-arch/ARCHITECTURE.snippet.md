## レイヤ境界（backend-clean-arch パック）
- 依存方向は**常に内向き**: `インフラ/アダプタ → ポート → コア`。コアはフレームワーク・DB・
  HTTP・UI を import しない。
- 境界の正確な定義は `gate.d/clean-arch.env`（CORE_DIRS / FORBIDDEN_IMPORT_REGEX）。
  L2 ガードが編集時に、gate（gate.d/）がマージ時に、同じ規則を機械検査する。
- 計画役へ: スライスは可能な限り「コア変更」と「アダプタ変更」を同一スライス内の縦割りで
  持たせる（層で水平に切らない — 水平スライスは全ワーカーの衝突点になる）。
